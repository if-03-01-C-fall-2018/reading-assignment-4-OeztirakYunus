CheatSheet 4

    Box Model

  1. Block elements have which kind of characteristics?
    A.: A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
    Example: <div>

  2. Inline elements have which kind of characteristics?
    A.: An inline element does not start on a new line and only takes up as much width as necessary.
    Example: <span>

  3. What is the margin of an Html element? Where can one find it?
    A.: The CSS margin properties are used to create space around elements, outside of any defined borders. With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).

  4. What is the padding of an Html element? How does it differ from margin?
    A.: The CSS padding properties are used to generate space around an element's content, inside of any defined borders. With CSS, you have full control over the padding. There are properties for setting the padding for each side of an element (top, right, bottom, and left). padding is the space between the content and the border, whereas margin is the space outside the border.

    Classes and Ids

  1. Write down a selector for a certain class
    A.: .intro

  2. Write down a selector for a certain html element of a certain class
    A.: <p> element with class = "hometown":
    p.hometown

  3. Write down a selector for a certain id
    A.: #firstname

  4. Write down a selector for a certain html element of a certain id
    A.: #myHeader - <h1 id="myHeader">My Header</h1>

  5. Write down the start tag of a certain html element with a specific class.
    A.: <p class = "city">

  6. Write down the start tag of a certain html element with a specific id.
    A.: <p id = "myHeader">

  7. Do classes select one or more elements?
    A.: More

  8. Do ids select one or more elements?
    A.: More

  9. How often can classes / ids occur on one page?
    A.: Classes : as many times as you want
    Ids: are only used once per page (you can use it more than once but it is not recommended)

    Floating

  1. Write down the css declaration to make an html element float to the left
    A.: float: left;

  2. Make an html element stop floating around another (for a specific direction, for all directions)
    A.: float: none;
